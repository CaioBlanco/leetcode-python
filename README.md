## 🧩 Técnicas Cobertas

- **HashMap** — busca e contagem em O(1)
- **Stack (Pilha)** — problemas de balanceamento
- **Two Pointers** — arrays ordenados e listas ligadas
- **Sliding Window** — substrings e subarrays
- **Prefix/Suffix** — produtos e somas acumuladas
